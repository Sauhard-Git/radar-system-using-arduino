# Arduino Radar System

## Overview
This project is a Mini Radar System developed using Arduino UNO, HC-SR04 Ultrasonic Sensor, and a Servo Motor. The system scans the surrounding area by rotating the ultrasonic sensor and detecting nearby objects in real time.

The project demonstrates the basics of embedded systems, sensor interfacing, and servo motor control.

---

## Features
- Real-time object detection
- Radar-style scanning mechanism
- Distance measurement using ultrasonic sensing
- Smooth servo motor rotation
- Beginner-friendly embedded systems project

---

## Components Used
- Arduino UNO
- HC-SR04 Ultrasonic Sensor
- Servo Motor
- Jumper Wires
- Breadboard
- USB Cable

---

## Working Principle
The servo motor continuously rotates the ultrasonic sensor between specific angles. The HC-SR04 sensor measures the distance of nearby objects and sends the data to the Arduino UNO for processing.

The scanning mechanism creates a radar-like detection system capable of identifying obstacles in its range.
